# איך מעלים אימג' לדוקר האב — מדריך צעד אחרי צעד

המטרה: לקחת אימג' שבנינו במחשב, ולהעלות אותו לאינטרנט,
כך שכל אחד בעולם יוכל להוריד ולהריץ אותו.

כל פקודה מופיעה בבלוק קוד בנפרד.
ההסבר בעברית תמיד מתחת, בשורה נפרדת.

---

## לפני שמתחילים

צריך חשבון בדוקר האב. נרשמים פעם אחת, בחינם, בכתובת הזו:

```text
https://hub.docker.com
```

בזמן ההרשמה בוחרים שם משתמש באנגלית, באותיות קטנות.
שם המשתמש הזה יהיה חלק מהכתובת של האימג', אז כדאי לבחור משהו קצר וברור.

במדריך הזה שם המשתמש לדוגמה הוא:

```text
almayo
```

החליפו אותו בשם המשתמש שלכם בכל מקום שהוא מופיע.

---

## שלב ראשון — להתחבר מהטרמינל

```bash
docker login
```

מחבר את הטרמינל לחשבון דוקר האב שלכם.
הטרמינל יבקש שם משתמש, ואז סיסמה.
אחרי הצלחה תופיע ההודעה שההתחברות הצליחה.

חשוב להבין: התחברות באתר בדפדפן לא מספיקה.
ההעלאה נעשית מהטרמינל, ולכן הטרמינל עצמו חייב להיות מחובר.

---

## שלב שני — לתת לאימג' שם מלא

```bash
docker tag users-management almayo/users-management:1.0
```

נותן לאימג' שם חדש שכולל את שם המשתמש שלכם.
זה הכרחי, כי דוקר האב מזהה כל אימג' לפי שם המשתמש של הבעלים.

פירוק השם המלא:
החלק הראשון הוא שם המשתמש שלכם.
החלק האמצעי הוא שם האימג'.
החלק שאחרי הנקודתיים הוא מספר הגרסה.

אפשר לתת גם תווית של גרסה אחרונה, כך שמי שלא מציין גרסה יקבל אותה:

```bash
docker tag users-management almayo/users-management:latest
```

נותן לאותו אימג' גם את התווית של הגרסה האחרונה.

---

## שלב שלישי — להעלות לאינטרנט

```bash
docker push almayo/users-management:1.0
```

מעלה את האימג' לדוקר האב.
דוקר מעלה את האימג' שכבה אחרי שכבה. בסוף תופיע הודעת סיום.

אם תייגתם גם גרסה אחרונה, מעלים גם אותה:

```bash
docker push almayo/users-management:latest
```

מעלה גם את התווית של הגרסה האחרונה.
השכבות שכבר עלו לא נשלחות שוב, אז זה מהיר.

---

## שלב רביעי — לוודא שזה עלה

נכנסים לכתובת של האימג' באתר. מחליפים את שם המשתמש בשלכם:

```text
https://hub.docker.com/r/almayo/users-management
```

אם רואים שם את האימג' — זהו, הוא חי באינטרנט וזמין לכולם.

---

## הצד השני — איך מישהו אחר מריץ את האימג' שלכם

מי שרוצה להריץ את האפליקציה שלכם, ממחשב אחר לגמרי, מקליד:

```bash
docker run -p 8080:3000 --env-file .env almayo/users-management
```

דוקר יוריד את האימג' מהאינטרנט לבד, ואז יריץ אותו.
לא צריך את הקוד המקורי, לא צריך גיט, לא צריך להתקין שום דבר ידנית.

הערה חשובה לגבי הסודות:
קובץ הסודות לא עולה לאינטרנט יחד עם האימג', וזה נכון ובטוח.
לכן מי שמריץ את האימג' צריך לספק קובץ סודות משלו.

---

## סיכום קצר — שלוש הפקודות

```bash
docker login
docker tag users-management almayo/users-management:1.0
docker push almayo/users-management:1.0
```

מתחברים. נותנים לאימג' שם מלא עם שם המשתמש. מעלים לאינטרנט.

---

## הרעיון בשורה אחת

```text
docker build  ->  IMAGE מקומי  ->  docker push  ->  IMAGE באינטרנט  ->  docker pull
```

בונים אימג' במחשב.
דוחפים אותו לדוקר האב.
מעכשיו כל אחד בעולם יכול למשוך אותו ולהריץ.
